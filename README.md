# 🐾 Pet Care

A responsive veterinary clinic website built with React ⚛️

## ✨ About The Project

Pet Care is a modern web application for a veterinary clinic where users can explore services, learn about pet treatments, and navigate through different pages with a clean UI.

## 🛠 Tech Stack

| Technology                   | Usage                        |
| ---------------------------- | ---------------------------- |
| React                        | Building UI                  |
| React Router DOM             | Page routing                 |
| Tailwind CSS v4              | Styling                      |
| Component-Based Architecture | Reusable components          |
| Vite                         | Fast development environment |

## 📌 Features

* Login / Logout pages
* Responsive design
* Separated pages & components
* Clean folder structure
* Modern UI with Tailwind CSS

## 🚀 Run Project

```bash id="0k6s90"
npm install
npm run dev
```

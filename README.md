# 🐾 PetCare Veterinary Clinic

A modern, responsive website for a veterinary clinic, built with React and TailwindCSS.


## 🛠️ **Tech Stack**

| Technology | Version | Purpose |
|------------|---------|---------|
| ⚛️ React | 18.x | Core library |
| 🧭 React Router DOM | 6.x | Routing management |
| 🎨 TailwindCSS | 3.x | Styling & design |
| ⚡ Vite | 5.x | Fast development bundler |
| 📦 Git & GitHub | - | Version control |
